1. Introduction: 

This is a rhythm game which simulates the Triko Drum Master game by enabling user to hit the drum with the keys on the keyboard. The beat is set to be 125ms.

There are 4 tracks corresponding to 4 controllers. They are controlled by up, down, left, right correspondingly. The controller can make voice.
